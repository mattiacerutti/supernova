# Supernova

> [!WARNING]
> **This project is WIP.**
>
> It is not yet ready for general use. Expect breaking changes, missing features, and rough edges.

Supernova is an **opinionated** AI coding GUI. It is built for speed and precision, with an obsessive attention to detail. Use it as a desktop app or from your browser.

Current tools in this space are either slow, unpolished, force you into a specific provider, or lack basic features. Supernova is designed to be fast and focused around its core principles:

- **Speed first.** No lag, even with insanely long sessions. The UI should never slow you down.
- **Details matter.** Every interaction should feel intentional, and just work. If it feels off, it needs to be fixed.
- **No bloat.** No novelty bloat, no flashy features, just the tools you need to ship faster.
